  sequenceDiagram

    server->>browser: HTML document
    deactivate server

    browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/main.css
    activate server
    server->>browser: .css file
    deactivate server

    browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/spa.js
    activate server
    server->>browser: .js file
    deactivate server

    note right of user: preload page HTML with css formatting
    note right of browser: browser executes .js file and fetches JSON from the server

    browser->>server: GET https://studies.cs.helsinki.fi/exampleapp/data.json
    activate server
    server->>browser: [{content: "alguma br aqui? :)",date: "2025-06-11T00:16:43.430Z"},...]
    deactivate server

    note right of browser: browser executes callback function and renders the notes
    browser->>user: display notes on screen

    note right of user: user inputs data into form fields
    user->>browser: form submit

    note right of browser: .js executes onsubmit function, adding the new input to the notes and reloading the notes component
    browser->>user: display notes on screen
    note right of user: user is happy it worked!

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa {content: "exercise 0.5", date: "2025-06-11T13-54:32.730Z"}
    activate server
    server->>browser: {"message":"note created"}
    deactivate server

    note right of user: user closes broswer tab
    deactivate browser
